Project Overview
This repository contains Power BI dashboards developed during my Data Analyst internship at To-Let Globe. The project focused on transforming raw data into interactive dashboards that help track social media performance, analyze property listing engagement, and monitor intern productivity through clear visual insights.

1. LinkedIn Performance Dashboard
To-Let Globe actively posted content on LinkedIn but lacked a structured way to monitor post performance. I designed an interactive Power BI dashboard that visualizes key metrics such as impressions, clicks, reactions, and shares. The dashboard enables the marketing team to track engagement trends, identify high-performing posts, and make data-driven content decisions.

2. Property Listing Dashboards (City-Wise)
The company promoted real estate listings across cities like Mumbai, Hyderabad, Gurgaon, and Kolkata but did not have a system to compare engagement across locations. I created separate dashboards for each city displaying metrics such as impressions, clicks, and reactions. These dashboards help the team understand location-specific performance and adjust their posting strategy accordingly.

3. Intern Performance Tracking Dashboard
Multiple interns were involved in content creation, which created a need for performance monitoring. I developed a KPI-based dashboard to track intern-wise metrics such as number of posts, total impressions, and engagement levels. This dashboard allows management to evaluate productivity, recognize high performers, and maintain accountability.

Tools Used:
Power BI

Data Visualization Techniques

Project Impact:
These dashboards helped the organization move from manual tracking to a structured, data-driven reporting system. The dashboards provide clear insights into social media engagement and team productivity while also serving as reusable tools for future analysis.
